Nombre del programa: play-porno
Autor: Rayo'scompany.
Descripción:
play porno es un reproductor de archivos de audio simple y accesible diseñado para ser utilizado con NVDA (aunque seguramente funcionará con jaws o voces tts como sapi 5. El programa reproduce archivos de audio en formatos MP3, WAV y OGG. Utiliza atajos de teclado para controlar la reproducción y es compatible con la salida de voz TTS para anunciar el estado y la información del reproductor.

Instrucciones de uso:
1. Ejecute el programa y verifique que la carpeta de archivos de audio esté configurada correctamente. Si no se ha configurado previamente, el programa le pedirá que seleccione una carpeta.
2. Una vez configurada la carpeta, el programa seleccionará aleatoriamente un archivo de audio para reproducir y comenzará la reproducción.
3. Utilice los siguientes atajos de teclado para controlar la reproducción:

    - Flecha izquierda: Retroceder 15 segundos
    - Flecha derecha: Avanzar 15 segundos
    - Barra espaciadora: Pausar / reanudar la reproducción
    - Q: Salir del programa
    - T: Anunciar el tiempo de reproducción actual y la duración total del archivo
    - Flecha arriba: Aumentar el volumen
    - Flecha abajo: Disminuir el volumen
    - N: Saltar al siguiente archivo aleatorio
    - 1-0: Avanzar al porcentaje correspondiente de la reproducción (10%, 20%, ..., 90%)
    - M: Silenciar / activar el sonido
    - F1: Mostrar el menú de ayuda con los atajos de teclado disponibles

4. Para cerrar el programa, presione la tecla "Q".

Requisitos:
- Python 3.10
- Pygame
- Librería VLC para Python
- Accessible_output2
